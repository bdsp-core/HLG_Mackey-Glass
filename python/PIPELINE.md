## Quick Start

### Reproduce the paper figures

```bash
cd python
uv run python -m scripts.run_paper_figures
```

Generates Figure 1 panels (per-segment EM fits) and Figure 2 (full-night overviews)
for all 4 example patients. Takes ~12 seconds.

### Run the full end-to-end pipeline from raw H5

```bash
uv run python -m scripts.run_end_to_end --study 7
```

Reads the raw 200Hz PSG, preprocesses to 10Hz, runs EM on all NREM segments,
and cross-validates against the pre-computed reference. Takes ~40 seconds per study.

### Run the EM algorithm on a study CSV

```bash
uv run python -m scripts.run_em_python --input data/paper_examples/HLG_OSA_Study_7.csv
```

### Generate all available publication figures

```bash
uv run python -m scripts.generate_all_figures
```

## Performance

Per 8-minute segment (4,800 samples at 10 Hz):
- **With Numba JIT**: ~0.7 seconds
- **Pure Python fallback**: ~16 seconds
- **MATLAB original**: ~3 seconds

Full study (44 segments): ~30 seconds with Numba, ~12 minutes without.

End-to-end pipeline (raw H5 в†’ all figures for 1 patient): ~40 seconds.
